### How to Test

* Open terminal
* Build project:
```jshelllanguage
./gradlew clean build   OR   ./gradlew.bat clean build
```
* Run command: 
```jshelllanguage
java -jar build/libs/reader-writer-poc-0.0.1-SNAPSHOT.jar
```
* Run the test with your IDE or:
```jshelllanguage
./gradlew test   OR   ./gradlew.bat test
```
